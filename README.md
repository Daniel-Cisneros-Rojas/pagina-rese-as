## Sistema de Formularios con PHPMailer
# Descripción
Este proyecto permite la creación de formularios web en PHP que envían los resultados directamente por correo electrónico utilizando la librería PHPMailer. Está diseñado para ser flexible y fácil de integrar en cualquier aplicación o sitio web, ofreciendo una solución práctica para la recolección de datos y su envío seguro vía email.

# Características
Creación de formularios dinámicos en PHP.

Envío de resultados por correo electrónico mediante PHPMailer.

Configuración sencilla de destinatarios y parámetros de envío.

Validación básica de campos para asegurar la integridad de los datos.

Compatible con servidores locales y de producción.

# Tecnologías utilizadas
Lenguaje: PHP

Librería: PHPMailer

Servidor: Apache / XAMPP
