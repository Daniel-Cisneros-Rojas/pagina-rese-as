## Sistema de Resenas y Notificaciones usando libreria PHPMailer

<img width="1742" height="778" alt="image" src="https://github.com/user-attachments/assets/ca3c1b31-4291-489b-a9fe-30323aabf3ef" />

Este proyecto fue desarrollado en PHP con base de datos en MySQL, integrando la librería PHPMailer para la gestión de correos electrónicos. El sistema permite a los usuarios registrarse y validar su cuenta mediante un código de activación único con tiempo de expiración, enviado directamente a su correo electrónico.

<img width="1634" height="788" alt="image" src="https://github.com/user-attachments/assets/6c768f2a-ae45-47ab-8bac-a5982edb4721" />


Una vez registrados, los usuarios pueden crear reseñas sobre películas, libros y series. Estas reseñas generan un espacio tipo foro donde otros usuarios pueden dejar comentarios, fomentando la interacción y el intercambio de opiniones.

<img width="862" height="871" alt="image" src="https://github.com/user-attachments/assets/62e02209-6e03-4618-9934-09cae29d9c8f" />

<img width="632" height="797" alt="image" src="https://github.com/user-attachments/assets/5502484c-7d80-4b02-b16f-f5ff6439abba" />


El sistema también incluye un mecanismo de notificaciones: cada vez que un usuario recibe un comentario en sus reseñas, se le envía un correo electrónico informando de la acción. De esta manera, se asegura una comunicación constante y efectiva entre los participantes de la plataforma.

<img width="1620" height="633" alt="image" src="https://github.com/user-attachments/assets/51f3329c-a722-4c05-a314-edc3c8d4af5d" />


Este enfoque combina autenticación segura, gestión de contenido y comunicación vía correo electrónico, ofreciendo una solución completa para la creación de comunidades en torno a reseñas culturales.
